Syntax — Quick reference

SET LEFT N
set left wheel speed to N (negative = reverse)N: −100 … 100 · default 0
SET RIGHT N
set right wheel speed to N (negative = reverse)N: −100 … 100 · default 0
WAIT T
wait T milliseconds before executing the next commandT: ms · min 1
STOP
stop both wheels immediately
SENSOR_LEFT
left sensor value (distance along left ray)returns: 0 – 500 · ∞ if nothing in range
SENSOR_RIGHT
right sensor value (distance along right ray)returns: 0 – 500 · ∞ if nothing in range
SENSOR_CENTER
center sensor value (distance along center ray)returns: 0 – 500 · ∞ if nothing in range
IF ...
if condition is true, execute the following commands
ELSE IF ...
if previous IF/ELSE IF was false, check this condition
ELSE
if none of the previous conditions match, execute these commands
#
comment — ignored by the robot (for humans)
END
end / loop
