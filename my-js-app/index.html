Syntax — Quick reference

SET LEFT N
set left wheel speed to N (negative = reverse)
SET RIGHT N
set right wheel speed to N (negative = reverse)
WAIT T
wait T milliseconds before executing the next command
STOP
stop both wheels immediately
SENSOR_LEFT
left sensor value (distance along left ray)
SENSOR_RIGHT
right sensor value (distance along right ray)
SENSOR_CENTER
center sensor value (distance along center ray)
IF ...
if condition is true, execute the following commands
ELSE IF ...
if previous IF/ELSE IF was false, check this condition
ELSE
if none of the previous conditions match, execute these commands
#
comment — ignored by the robot (for humans)
END
end / loop
< > =
operators: < (less than), > (greater than), = (equal)
